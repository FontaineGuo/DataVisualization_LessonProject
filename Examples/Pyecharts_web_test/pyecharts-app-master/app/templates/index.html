pyecharts

关于项目

pyecharts 是一个用于生成 Echarts 图表的类库。Echarts 是百度开源的一个数据可视化 JS 库。用 Echarts 生成的图可视化效果非常棒，为了与 Python 进行对接，方便在 Python 中直接使用数据生成图，我写了这个项目。

本网站只是用于展示使用 pyecharts 生成的图的效果，如果您对项目感兴趣，可前往 pyecharts/pyecharts 了解更多内容

BTW: 这个网站是用 Flask+pyecharts 写的，并部署在 Heroku。本网站源码在 pyecharts/pyecharts-app
